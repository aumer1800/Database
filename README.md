Student Data Management System

A backend-based Student Data Management System developed using Node.js, Express.js, and MongoDB.
This project performs CRUD (Create, Read, Update, Delete) operations for managing student records through REST APIs.

Features
Add new students
View all student records
Update student information
Delete student records
MongoDB database integration
REST API implementation
Technologies Used
Backend
Node.js
Express.js
Database
MongoDB
Mongoose
Tools
Postman
Nodemon
dotenv
